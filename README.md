## So Long - A 2D Game Project

**Description:**  
This project is a very small 2D game. It is built to make you work with textures, sprites, and introduce some very basic gameplay elements.

**Usage:**  
1. Compile the game:  

```make```

2. Run the game:  

```/so_long map [MAP_FILE.ber]```
